# NLPtools-Library
個人的によく使うものをライブラリとしてまとめました  
随時更新していきます  
ライブラリ公開は初めてなのでご指摘、不明な点があれば連絡よろしくお願いします

## インストール方法 
`pip install git+https://github.com/ganbon/NLPtools-Library`を実行
## ライブラリ機能一覧
- 形態素解析(morpheme.py)
- 複数のファイル形式のファイル保存、ロード(file_system.py)
- 文字の前処理(norm.py)
- スクレイピング(text_get.py)
